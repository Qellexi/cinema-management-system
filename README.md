# cinema-management-system
Cinema Management System — це вебзастосунок для управління кінотеатром та онлайн-бронювання квитків.  Проєкт розробляється в межах SoftServe Practice і демонструє повний цикл розробки: від проєктування бази даних та архітектури до реалізації бізнес-логіки, авторизації та користувацького інтерфейсу.
