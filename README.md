# Cinema Management System

Cinema Management System — це вебзастосунок для управління кінотеатром: фільмами, сеансами, залами та бронюванням місць.

Проєкт реалізований на **ASP.NET Core MVC** з використанням стандартної архітектури MVC.

---

## Основні можливості

- Управління фільмами
- Управління залами кінотеатру
- Створення та редагування сеансів
- Перегляд розкладу
- Бронювання місць
- Адміністративна панель для керування даними

---

## Технології

- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Razor Views

---

## Структура проєкту

CinemaApp/
├── Controllers/
├── Models/
├── Views/
├── Data/
├── wwwroot/
└── Program.cs

## Основні розділи системи

### Movies
- Додавання фільмів
- Редагування фільмів
- Видалення фільмів

### Halls
- Створення залів
- Налаштування кількості місць

### Sessions
- Перегляд розкладу сеансів
- Прив’язка сеансу до фільму та залу


---
